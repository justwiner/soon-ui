### AutoScroll Attributes

| 参数          | 说明            | 类型            | 可选值                 | 默认值   |
|-------------  |---------------- |----------------|----------------------  |-------- |
| direction     | 移动方向         | String         | `bottom`，`top`，`right`，`left`  | top     |
| hStop         | 鼠标放入容器中时，是否停止动画     | Boolean          |    —     | false |
| speed         | 每秒移动的距离，单位px     | Number          |  —                    | 70    |


### AutoScroll Methods

| 事件名         | 说明            | 参数            |
|-------------  |---------------- |---------------- |
| start         | 开始轮播         | —              |
